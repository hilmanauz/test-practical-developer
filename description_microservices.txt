Flow app frontend get data to backend
pada sisi frontend ketika hit API KATALOG, user yang akses harus memiliki token terlebih dahulu untuk mendapatkan data yang diinginkan sesuai dengan API yang di hit, ketika user tidak memiliki token maka app akan mengembalikan pesan error dan tidak mendapatkan akses ke database. Jika user memiliki token maka user berhasil mendapatkan akses ke database sesuai dengan API yang di-hit. Sisi Backend menggunakan teknologi yang bernama docker untuk membantu implementasi microservices dimana pada setiap port hanya terdapat satu database yang bisa diakses, maka jika ada salah satu port yang error atau down maka database yg ada diport itu saja yang tidak bisa diakses dan tidak mengganggu database yang lainnya
